# Scanalyze

### AI-Powered Resume Analysis Platform

Analyze resumes, compare them against job descriptions, generate ATS compatibility scores, and receive actionable AI-powered feedback to improve job applications.

---

## 📖 Introduction

Scanalyze is an AI-powered resume analysis platform designed to help job seekers optimize their resumes for modern Applicant Tracking Systems (ATS).

Users can upload resumes, compare them against job descriptions, receive ATS compatibility scores, and get personalized AI feedback to improve their chances of securing interviews.

Built with React, TypeScript, React Router, Tailwind CSS, Zustand, and Puter.js, Scanalyze delivers a fast, scalable, and responsive user experience.

## 🚀 Quick Start

### Clone the Repository

```bash
git clone https://github.com/vanshhchauhan/scanalyze.git
cd scanalyze
```

### Install Dependencies

```bash
npm install
```

### Run the Development Server

```bash
npm run dev
```

Open http://localhost:5173 in your browser.
